transition,过渡 ；@keyframes，创建动画

鼠标悬停实现对width的过渡操作
transition: width (方式)1s(时间) linear(时间曲线) 2s开始时间 ;
注：@keyframes 规则用于创建动画。在 @keyframes 中规定某项 CSS 样式，就能创建由当前样式逐渐改为新样式的动画效果。

@keyframes 动画名称{
0% {top:0px; background:red; width:100px;}
100% {top:200px; background:yellow; width:300px;}
}

animation执行动画

div{
animation: myfirst 5s linear 2s infinite alternate;

}
